# Návod jak si udělat svůj první projekt v nestu

## Instalace

1. Nejdříve si stáhneme [Node.js](https://nodejs.org/en)

- Návod na instalaci Node.js [zde](./00-node.mdx)

2. Nainstalujeme si nest

- otevřeme `CMD`
- příkaz pro globalní instalaci nestjs

```
npm i -g @nestjs/cli
```

## Založení projektu

3. Založíme si nový projekt s názvem `kebab` 🥙

- pomocí `cd` se dostaneme do složky kde chceme vytvořit projekt

```
cd C:\Bagros\Code\api
```

```
nest new kebab
```

4. Zapneme si server nestu
   Tento příkaz zapne server na vašem počítači, který vám odevzdává JSON (api server)

```
npm run start:dev
```
