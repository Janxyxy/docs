# Tutorial jak si udelat svuj prvni projekt v nestu

## Instalace

1. Nejdříve si stáhnem [node.js](https://nodejs.org/dist/v16.15.1/node-v16.15.1-x64.msi)

2. Nainstalujeme si nest

- otevřeme `CMD`
- příkaz pro globalní instalaci nestjs

```
npm i -g @nestjs/cli
```

## Založení projektu

3. Založíme si nový projekt s názvem `kebab` 🥙

- pomocí `cd` se dostaneme do složky kde chceme vytvořit projekt

```
cd C:\Bagros\Code\api
```

```
nest new kebab
```

4. Zapneme si server nestu
   Tento prikaz zapne server na vasem pocitaci, který vám odevzdává JSON (api server)

```
npm run start:dev
```
