## Transportní vrstva

Dělíme na `TCP` A `UDP`

### TCP

- Zasílá segment s kontrolou (kontroluje jestli paket dorazil)
- Kvůli kontrole je `pomalejší`
- Kontrola:

1. Host 1 - Zeptá se jestli tam je Host 2 (synchronizace)
2. Host 2 - Odpovídá, že tam je (accept), a zašle ještě jednu otazku jestli tam je (synchronizace)
3. Host 1 - Odpovádá, že tam je (accept), a pošle segment
   (opakuje se to dokola)

### UDP

- Zasílá `datagram` bez kontroly
- je to rychlejší ale může vzniknout ke ztrátám (dobré u videohovorů)
