# Rozdělení sítí podle rozsahu

## PAN

- Osobní počítačová síť
- Maličká síť v rámci jedné místnosti se 2 nebo 3 zařízeními
  > - _bluetooth_ připojení mezi dvěma zařízeními

## LAN

- Lokální počítačová síť
- Správa LAN je většinou svěřena 1 správci.
  > - nejčastěji využívaná v několika místnostech, v jedné budově.

## MAN

- Metropolitní počítačová síť
- Několik menších LAN sítí
- má svého správce sítě
  > - propojována opticky
  > - univerzity, organizace

## WAN / GAN

- Rozsáhlá počítačová síť
  > - WAN síť tvoří páteřní vysokorychlostní optické nebo satelitní sítě
  > - Internet

### Internet

Jedná se o celosvětový systém propojených počítačových sítí („síť sítí“).

- TCP/IP

### Intranet

- Jedná se označení pro část počítačové sítě, která používá stejné technologie jako Internet (rodinu protokolů TCP/IP, přenosový protokol HTTP atp.).
- Na rozdíl od internetu je však intranet privátní („soukromý“), tj. jeho využívání je omezeno na malou skupinu uživatelů (například pracovníci firmy, školy).

## Fyzická

- popisuje reálné propojení hostů v síti

# Podle způsob komunikace

### Klient-server

- Server poskytuje uživatelovi službu

### Peer-to-peer

- Službu si poskytují hosti
- Torrenty => Host 1 odesílá soubory, Host 2 stahuje soubory

# Bezpečnost

## Viry

- Virus, červ, Malware

## Hrozby

- DDoS utok = přetížení sítě
- Phishing stránka = falešná stránka, do které dáme udaje a utočník ukradne data

:::warning Virus vs. Červ
Virus se přichití k něčemu (textovej soubor, ...), červ se šíří sám.
:::
