# Sociální a zdravotní pojištění

## Sociální pojištění
- důchodové pojištění.
- nemocenské pojištění => nemocenské, mateřské, otcovská, OČR
- příspěvek na státní pojistku zaměstnanosti
- sazby sociálního pojištění
  - zaměstnavatel platí **24,8%**
  - zaměstnanec platí **7,1%**

## Zdravotní pojištění
- základní zdravotní péče
- sazby zdravotního pojištění
  - zaměstnavatel platí **9%**
  - zaměstnanec platí **4,5%**
