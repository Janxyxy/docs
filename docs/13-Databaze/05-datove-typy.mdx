# Datové typy

## Celočíselné datové typy

| Datový typ |     Rozsah (unsigned)     |
| :--------: | :-----------------------: |
|  Tinyint   |  $-2^{7}$ až $2^{7} - 1$  |
|  Smallint  | $-2^{15}$ až $2^{15} - 1$ |
| Mediumint  | $-2^{24}$ až $2^{24} - 1$ |
|    Int     | $-2^{32}$ až $2^{32} - 1$ |
|   Bigint   | $-2^{63}$ až $2^{63} - 1$ |

## Speciální číselné datové typy

**BIT** -> $0$ nebo $1$ <br/>
**DECIMAL** -> Zapisujeme jako proměnnou, s určeným počtem číslic a desetinných míst

```sql
decimal(<počet desetinných míst> ,<počet čísel>)
```
